# User Stories - Zamalek Store

## 1. Customer Stories

### Browsing & Discovery

- **As a Customer**, I want to view a list of product categories so that I can find the type of item I'm looking for.
- **As a Customer**, I want to search for specific products by name so that I can quickly locate items.
- **As a Customer**, I want to click on a product to see its details, price, and images so that I can make an informed buying decision.

### Cart & Purchasing

- **As a Customer**, I want to add items to my shopping cart so that I can purchase multiple items at once.
- **As a Customer**, I want to view my cart and adjust quantities or remove items before checkout.
- **As a Customer**, I want to pay securely using my credit card (via Paymob) so that my transaction is safe.
- **As a Customer**, I want to receive an order confirmation email so that I have a record of my purchase.

### Account Management

- **As a Customer**, I want to create an account so that I can save my shipping address for future purchases.
- **As a Customer**, I want to view my past orders so that I can track my spending and order status.

## 2. Admin Stories

### Product Management

- **As an Admin**, I want to add new products with images, descriptions, and prices so that they are available for sale.
- **As an Admin**, I want to update product stock levels so that customers don't buy out-of-stock items.
- **As an Admin**, I want to create and manage product categories to keep the store organized.

### Order Management

- **As an Admin**, I want to view a list of all incoming orders so that I can process them.
- **As an Admin**, I want to change the status of an order (e.g., from "Processing" to "Shipped") so that the customer is updated.
- **As an Admin**, I want to see the payment status of each order to ensure I've been paid before shipping.

### Dashboard

- **As an Admin**, I want to see a summary of daily/weekly sales so that I can track business performance.
